Welcome to Number Drop

Math is fun!!


About the game

Let’s drop some numbers! In this fun logic game, you must carefully land matching number blocks next to one another to merge them into a single block. Keep merging blocks to free up space. Run out of space and it’s game over!

How to Play the game

Play the game by selecting one of the columns to capture the falling number tile. If the tile falls adjacent to another tile having the same number, they become one tile with the numbers added. Repeat this for every falling tile until you get the largest number, which is 2048 in this game. When you have a 2 tile, drop it to be adjacent to another 2 tile, so that the numbers combine to become one 4 tile. This will earn you 4 Points. Then try aligning a falling 4 tile to be adjacent to another 4 tile to make a 8 tile. This will earn 8 points and so on. Your objective is to add together the numbers until you reach the highest number, without running out of space. At anytime, you can restart the game with a new set of numbers. Enjoy the Number drops!
